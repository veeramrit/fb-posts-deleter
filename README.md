# FB Posts Deleter

A free Chrome extension to delete all your Facebook Posts at once automatically — 
no paid tools, no subscriptions, no login required.

## 🔗 Full Guide & Download
👉 [stacknix.dev/how-to-delete-all-facebook-posts-at-once](https://stacknix.dev/how-to-delete-all-facebook-posts-at-once/)

## How to Install

1. Download this repository as ZIP (green Code button → Download ZIP)
2. Extract the ZIP on your computer
3. Open Chrome → go to `chrome://extensions`
4. Enable **Developer Mode** (top right toggle)
5. Click **Load unpacked** → select the extracted folder
6. Open Facebook Activity Log and click the extension icon

## How to Use

1. Go to your Facebook Activity Log filtered to Posts:
   `facebook.com/profile.php?sk=allactivity&category=TIMED_PUBLISHED_STORIES`
2. Click the FB Posts Deleter extension icon
3. Click **⚡ Delete ALL Posts**
4. Watch the counter go up as posts are deleted automatically

## Features

- ⚡ Delete ALL posts at once via Activity Log
- 🔍 Debug scan for troubleshooting
- ⬛ Stop anytime
- 🔗 Stacknix.dev branding with clickable link
